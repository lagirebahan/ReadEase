To run the program:
1. create database 'readease' and import 'readease.sql'
2. npm install in /backend folder
3. run backend with 'npm run dev'
4. open /lib/config/config.dart and insert ipv4 address
5. 'flutter clean' in /frontend
6. 'flutter pub get' in /frontend
7. 'flutter run' while connected to an android device in /frontend
